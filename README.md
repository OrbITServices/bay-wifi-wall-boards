# Bay WiFi Wall Boards - Roles Version

Default logins:

Admin: admin / changeme
Editor: editor / changeme

Install:

```bash
unzip bay-wifi-wall-boards-roles.zip
cd bay-wifi-wall-boards-roles
cp .env.example .env
nano .env
npm install
npm start
```

Admin: http://PI-IP:3000/admin
